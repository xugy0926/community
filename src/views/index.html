持续学习

初学者的常见问题



从来没有学过编程，应该从什么编程语言开始？

每种编程语言都有其自身特点，解决的问题以及运用场合也不一样。如果你学习编程是想找到一份工作，学习 JavaScript 是蛮好的选择。
相对于其他语言，JavaScript 入门更简单一些，而且你能运用极少的 JavaScript的知识就能开发前端页面，从而使你成为一名前端工程师。

我应不应该报一个线下的培训学校？

线下的培训学校参差不齐，对学校的选择要谨慎一些。如果有急于找工作的压力，并且有相对靠谱的培训学校，就可以去试试。
如果对学费有顾虑，或者想挑战一下自己的自学能力，你可以在社区获得大家的帮助。你有问题大家都愿意帮助你。

已经工作 5、6 年了，想转型学编程，能成功吗？

同样学习编程，刚毕业的学生会更有优势，公司在招聘的时候也更倾向招年轻的。 学习编程这事是越早越好。如果自己有坚定的信念，就应该开始执行。

现在开始学习 JavaScript，并成为一名前端工程师，大概多长时间？

3 - 5个月足以让你找到一份初级工程师的工作。JavaScript 的难度并不大，关键是怎么找到一条最佳路径去学习。
获取最佳路径的方式有很多种，找一个靠谱的培训学校，报名一个靠谱的线上班，加入一个社区，获得社区成员的帮助。
